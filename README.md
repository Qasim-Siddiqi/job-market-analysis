#  Job Market Analysis (Naukri Dataset)

##  Overview

Analyzed real-world job market data to uncover insights about salaries, experience, industries, and in-demand skills using Python.

---

##  Tech Stack

* Python (Pandas, NumPy)
* Matplotlib, Seaborn

---

##  Key Work

* Cleaned messy salary and experience data
* Created features: `avg_pay`, `avg_exp`
* Handled missing values
* Converted raw text data into structured format

---

##  Key Insights

* Salary increases with experience
* Certain industries offer higher median salaries
* A few companies dominate job postings
* Specific skills are linked to higher-paying jobs

---

##  Visualizations

* Salary distribution plots
* Correlation heatmap
* Industry-wise salary comparison
* Skills vs salary analysis

---
